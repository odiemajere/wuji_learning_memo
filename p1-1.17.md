# 赋值运算符

|             |              |                |
| :---------: | :----------: | :------------- |
|     a=b     |     a=b      | 基本赋值运算符 |
|    a+=b     |    a=a+b     | 加赋值         |
|    a-=b     |    a=a-b     | 减赋值         |
|    a\*=b    |    a=a\*b    | 乘赋值         |
|    a/=b     |    a=a/b     | 除赋值         |
|    a%=b     |    a=a%b     | 求余赋值       |
|    a<<=b    |    a=a<<b    | 左移赋值       |
|    a>>=b    |    a=a>>b    | 右移赋值       |
|    a&=b     |    a=a&b     | 按位与赋值     |
| a &#124; =b | a=a &#124; b | 按位或赋值     |
|   a\*^=b    |   a=a\*^b    | 按位异或赋值   |

## 按位运算范例

```C
unsigned char a;
a = 10;
a <<= 1;
//a = a<<1;
//a = 0b 0000 1010 = 0x10
//a<<1=0b 0001 0100 = 0x14
```

```C
unsigned char a;
a = 10;
a >>= 1;
//a = a>>1;
//a = 0b 0000 1010 = 0x10
//a>>1=0b 0000 0101 = 0x05
```

```C
unsigned char a;
a = 10;
a &= 3;
//a = a&3;
//a = 0b 0000 1010 = 0x10
//3 = 0b 0000 0011
//a&3=0b 0000 0010 = 0x02
```
