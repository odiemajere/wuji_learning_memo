# 转义字符

| 原字符 | 十进制 | 转义字符 |   描述   |
| :----: | :----: | :------: | :------: |
|   /    |  &#47  |   &sol   | 斜线符号 |
|   #    |  &#35  |   &num   | 数字标志 |
|   –    | &#8211 |  &ndash  |   减号   |
|   ÷    | &#247  | &divide  |   除号   |
|   \|   | &#124  | &verbar  |   竖线   |
| &#124; | &#124  | &verbar  |   竖线   |
