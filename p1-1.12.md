# switch语句
##格式
```C
switch(变量表达式){
    case 常量1:{
        语句1
    }
    break;
    case 常量2:{
        语句2
    }
    break;
    case 常量3:{
        语句3
    }
    break;
    case 常量4:{
        语句4
    }
    break;
    default:{
        语句5//变量不等于以上列出的常量
    }
}
```
