# C语言关系运算符
## C语言的10类运算符
    1. 算术运算符
    2. 关系运算符
        - <     小于
        - <=    小于等于
        - >     大于
        - >=    大于等于
        - ==    等于
        - !=    不等于
        - 关系运算符的运算结果为真（1）或假（0）
        - 关系运算符一般与if、while语句配合使用
    3. 逻辑运算符
    4. 位操作运算符
    5. 赋值运算符
    6. 条件运算符
    7. 逗号运算符
    8. 指针运算符
    9.  求字节数运算符
    10. 特殊运算符
