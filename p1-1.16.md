# 逻辑运算符

|     符号     |  运算  |
| :----------: | :----: |
|      &&      | 与运算 |
| &#124;&#124; | 或运算 |
|      !       | 非运算 |

## 与运算范例

```C
void main(void)
{
	unsigned char i;
	//与运算   if   while
	unsigned a,b,c;
	a = 1;
	b = 2;
	c = 3;

	LED1 = 0;
	LED2 = 0;

	if((b>a)&&(b<c))
	{
		LED1 = 1;
        LED2 = 1;
	}
	while(1)
}
```

## 或运算范例

```C
void main(void)
{
	unsigned char i;
	//与运算   if   while
	unsigned a,b,c;
	a = 6;
	b = 2;
	c = 3;

	LED1 = 0;
	LED2 = 1;

	if((b>a)||(b<c))
	{
		LED1 = 1;
        LED2 = 1;
	}
	while(1)
}
```
